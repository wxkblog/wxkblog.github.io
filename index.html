Git

我想把我hexo上的第一篇博客写成如果搭建博客，感谢这个平台。
我们分三步走

- GitHub的申请，git的下载,创建库
- Node.js、hexo的下载安装
- 博客的搭建

GitHub的申请，git下载安装

运行git bash
设置git的user name和user email

生成SSH密钥

1.查看是否已经有了ssh密钥：cd ~/.ssh
如果没有密钥则不会有此文件夹，有则备份删除
2.生成SSH密钥：
$ ssh-keygen -t rsa -C "你的email"
按3个回车，密码为空。

最后得到了两个文件：id_rsa和id_rsa.pub

3.添加密钥到ssh：ssh-add 文件名
需要之前输入密码。
4.在github上添加ssh密钥，这里添加的是“id_rsa.pub”里面的公钥，保存的位置上面显示了
在github的添加ssh密钥的界面上复制粘贴就OK了。
登陆进去后可以看到,点setting，点SSH and GPG keys,New SSH key,title可以随便填，把密钥复制进去,保存


5.创建库，Your profile->repositories->new->
在Repository name里面填： 你的用户名.github.io 创建。暂时不用管这里了。

Node.js、hexo的下载安装

先下载Node.js
打开git bash，安装hexo
利用npm命令安装。这里可能回车之后没反应，需要等一会儿才会出来东西，不要急躁
npm install -g hexo

创建hexo文件夹

安装完成后，在一个你喜爱的文件夹（如D:\hexo）下执行以下指令(在 D:\hexo内点击鼠标右键，选择git bash)，Hexo即会自动在目标文件夹建立网站所需要的所有文件。
hexo init

安装依赖包

npm install

本地查看

现在我们已经搭建起本地的hexo博客了，执行以下命令(在D:\hexo)，然后到浏览器输入localhost:4000看看。
hexo generate
hexo server
本地博客已经搭建起来了，别人看不到的。下面，我们要部署到Github让别人也能看到。
